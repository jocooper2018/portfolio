Suika

Compétence associé : Réaliser un développement d’application

Entre les cours, j’ai vu des étudiants jouer à un jeu : Suika Game, le jeu de la pastèque. J’ai trouvé le concept du jeu intéressant. J’ai donc essayé de refaire le jeu. Lors de ce projet, je me suis concentré sur la partie fonctionnelle du jeu. Les visuels ne sont donc pas très poussé.

Le concept du jeu est simple  Une boite dans laquelle on peut faire tomber des fruits. Le triangle représente l’endroit d’où on va faire tombé le prochain fruit. On peut le déplacé avec la souris ou le clavier pour choisir où faire tomber le fruit. Les fruits sont représenté par les cercles de couleur.

La ligne verticale en dessous du triangle permet de voir où le fruit va tomber. Le fruit peut être lâcher en cliquant ou en appuyant sur le touche espace.

Lorsqu’un fruit touche un autre fruit identique, les deux fruits fusionne pour donner un autre fruit plus gros.

L’objectif du jeu est de faire le plus de combinaisons de fruit possible sans faire déborder la boîte. Lorsqu’un fruit touche la barre du haut pendant trop longtemps, la partie s’achève.

Le projet à été réaliser avec le logiciel Unity.

Le code source est en C#.
