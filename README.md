## Quick start

### Prerequisites

- [Node.js](https://nodejs.org/)
- [yarn](https://yarnpkg.com/)

### Install

```bash
git clone <repo-url>
cd portfolio
yarn
```

### Run development server

```bash
yarn dev
```

### Build for production

```bash
yarn build
```

### Preview production build

```bash
yarn preview
```

## Deployment

```bash
yarn deploy
```
